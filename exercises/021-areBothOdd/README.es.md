# `021` areBothOdd

## 📝 Instrucciones:

1. Escribe una función llamada `areBothOdd`. Que dados 2 números, `areBothOdd` retorna `true` i ambos números son impares y `false` si ambos números no lo son.

## Ejemplo de Entrada:

```js
var output = areBothOdd(1, 3);
console.log(output); // --> true
```

## Ejemplo de Salida:

```bash
true
```
